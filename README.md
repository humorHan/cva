## @humorhan/cva

cva(create vue app)是基于webpack封装的脚手架，旨在帮助FE快速创建一个vue2或vue3工程，并提供统一、稳定、可拓展的前端构建服务。

### 安装 `@humorhan/cva`

`yarn add @humorhan/cva`

### 初始化仓库

`cva-tools create [project name]`

### 本地启动

`cva serve`

### 部署环境构建

`cva build`
